Amazon Vine Canada Eligible items helper

The extension is now active and you should see a new toolbar above the items in the vine listings.

The community can now cast votes to signify if an item as import fees. Once confirmed, the item will be automatically marked.

Privacy notice: While the server handling the votes receive your IP address, the IP gets hashed and only that hash value is stored to avoid double voting. No futher information about your identity is stored in the database.
